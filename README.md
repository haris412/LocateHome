# LocateHome
A modern web platform that allows users to buy or rent properties easily.
The system supports multiple user roles and provides a seamless experience for property listing, searching, and management.

This platform enables:

Users to browse properties for Sale or Rent
Advanced filtering and search
Property listing management
Secure authentication & role-based access
Admin moderation and platform management


User Roles
1️ Guest User

Browse properties
Filter by location, price, type, rent/sale
View property details
Register / Login

2️ Registered User (Buyer / Renter)

All guest permissions
Save favorite properties
Contact property owners/agents
Manage profile
Track inquiries

3️ Property Owner / Agent

Create property listings (Buy or Rent)

Upload property images
Edit and delete listings
View inquiries from users
Manage property status (Available / Sold / Rented)

4️ Admin

Manage users
Approve / Reject property listings
Remove inappropriate content
Monitor platform activity
Manage categories, property types, and locations
